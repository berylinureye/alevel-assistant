讲解重点 / Speaker Notes

实线是学生当次会等待的主流程：上传、识别、匹配、批改、校验、返回结果、继续练习。

置信度节点不是第二次批改，而是把批改结果分成“可直接展示”“需要复核提示”“不要硬猜”几种表达方式。

虚线是异步质量闭环：记录延迟、准确率、复核率和覆盖边界，用来优化后续版本，不阻塞当次结果。
